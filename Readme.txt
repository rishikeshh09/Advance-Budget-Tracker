#Overview
Advance Budget Tracker is a comprehensive web application designed to help users manage and track their personal finances. The application allows users to categorize their income and expenses, set budgets, and visualize their financial data through interactive charts. This project was developed as part of a college assignment and showcases a variety of web development techniques and tools.

Features
User Authentication: Secure login and registration system.
Dashboard: Overview of financial status with quick access to key metrics.
Budget Management: Create, edit, and manage multiple budgets.
Expense Tracking: Categorize and record daily expenses.
Income Tracking: Record and manage sources of income.
Reports & Analytics: Visualize income, expenses, and budget over time using charts and graphs.
Responsive Design: Works seamlessly across desktops, tablets, and mobile devices.
Notifications: Get alerts when budgets are exceeded or bills are due.
Technologies Used
JavaScript (65.0%): Core logic for dynamic features, including form validations, data handling, and UI interactions.
HTML (18.1%): Structuring the web pages and content.
CSS (8.1%): Styling the application for a visually appealing and responsive design.
PHP (6.1%): Server-side scripting for managing user sessions, processing forms, and interacting with the database.
SCSS (2.6%): Advanced styling with variables and mixins for better maintainability.
Shell (0.1%): Scripts for automating deployment and setup tasks.
